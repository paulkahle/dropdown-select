This is a mutiselect dropdown with built-in funtionality for selecting and deselecting all options. The all option is completely managed by the component itself.

Treatment of Empty Selected Values

Sometimes you may want to treat an empty value as "All". Setting `emptyValueMode="all"` will show all selected if `value` is empty and set value to an empty array when all options are selected. If no options are selected `value` will contain a single value determined by the `noneSelectedValue` property which defaults to -1.

Trigger Events

Set an openTrigger and closeTrigger to control if the dropdown opens on hover or click. The default openTrigger is 'click' and closeTrigger is 'hover'.

Open=Click, Close=Hover (default)
Open=Click, Close=Click
Open=Hover, Close=Hover
Open=Hover, Close=Click

Hiding the "All" option

You can hide the "All" option by setting showAllOption=false
